# OMT Stock Transport Order Request

## Module Features

- **Transport Request Management**  
    Create, edit, and track stock transport requests between warehouses or locations.

- **Request Approval Workflow**  
    Supports multi-level approval process for transport requests.

- **Automated Stock Moves**  
    Automatically generates internal transfers upon request approval.

- **Customizable Request Types**  
    Define and manage different types of transport requests as per business needs.

- **Request Status Tracking**  
    Monitor the status of each request (Draft, Waiting Approval, Approved, Done, Cancelled).

- **User Access Control**  
    Restrict request creation, approval, and validation based on user roles.

- **Integration with Inventory**  
    Seamless integration with Odoo Inventory for real-time stock updates.

- **Notifications & Activity Log**  
    Automated notifications and activity logging for request lifecycle events.

- **Reporting**  
    Generate reports on transport requests, including pending, approved, and completed transfers.

- **Audit Trail**  
    Full traceability of all actions performed on transport requests.

- **Custom Notes & Attachments**  
    Add notes and attach documents to transport requests for better communication.
